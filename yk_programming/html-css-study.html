HTML,CSSの勉強

　ブログを書きつつ、HTMLも一緒に勉強すれば、一石二鳥になると思いましたが、手作り感がすごいです。
　やっぱり、他の人が作った優れたフレームワークを使った方がいいかもしれません。

HTMLの例


<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <link rel="stylesheet" href="./style.css">
    <title>タイトル</title>
</head>
<body>

</body>
</html>

よく使うタグ


<h1>見出し（大きい）</h1>
......
<h6>見出し（小さい）</h6>
<p>段落、文章</p>
<a href="URL">リンクを貼る</a>
<br> 改行
<hr> 線
<b></b> 太文字
<div>
    一つのかたまり
</div>
<article>
    大きい記事
</article>
<section>
    一区分、小記事
</section>

CSSについて（CSSはまだよくわかりません。）


ファイルの最初に書く
@charset "utf-8"; 

タグの設定
body {
    font-family: sans-serif;
    color: #FFFFFF;
    background-color: #EEEEEE;
    line-height: 1.6;
}

クラスの設定
.className {
    padding: 10px;
    text-align: center;
}


CSS難しいです。
暇つぶし程度に勉強していきます。
